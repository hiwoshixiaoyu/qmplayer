# qmplayer
 基于ffplay、C++11开发的跨平台视频播放器。旨在把解码和渲染播放进行剥离，方便进行移植。

# 如何编译
 windows下载ffmpeg

 下载网址：https://github.com/BtbN/FFmpeg-Builds/releases

- 添加include到如下文件夹
\qmplayer\ffplay\include
- 添加.lib  libswresample.dll.到如下文件夹
\qmplayer\ffplaylib


# 选择构建套件 
minggw64编译即可
